## Overview
While many websites pass basic accessibility tests, they often remain difficult or impossible for visually impaired users to actually navigate. Current automated auditing tools only catch between 14% and 38% of the accessibility barriers that screen reader users face. 

This project bridges that gap by introducing an AI agent that navigates web interfaces using only screen reader input and output, without relying on any visual information like screenshots. By mimicking the sequential, audio-based navigation of a human user, the AI acts as a practical diagnostic tool to uncover real-world navigation failures that traditional static tools cannot detect.

## Key Features
* **Screen-Reader Only Navigation:** The agent relies entirely on text-to-speech output to build a mental model of the interface, proving that visual layouts aren't necessary for AI web navigation.
* **Advanced Accessibility Auditing:** The system goes beyond standard compliance checklists to evaluate if a user can actually complete complex, multi-step goals.
* **AI-Powered Reasoning:** Utilizes Large Language Models (LLMs) to interpret ambiguous website labels and adapt to dynamic web pages.
